# AU Agent — Format แจ้งแผนขนส่ง

> ไฟล์นี้เป็น spec ของ format ข้อความที่ส่งมาทาง LINE เพื่อแจ้งแผนขนส่ง
> เมื่อ format เปลี่ยน → แก้ที่นี่ก่อน แล้ว implement regex ใน `parsePlanText()` ตาม

---

## Format มาตรฐาน (ปัจจุบัน)

```
ชื่อลูกค้า : Betagro ลพบุรี
วันที่จัดส่งลูกค้า : 16/6/2026
สินค้า : Wood chips ไม้สับ
จำนวนพ่วง : 2
```

---

## Field Definitions

| Field ใน LINE | Field ใน DB | หมายเหตุ |
|---|---|---|
| `ชื่อลูกค้า` | `customer` | ชื่อโรงงาน/ลูกค้าปลายทาง |
| `วันที่จัดส่งลูกค้า` | `ship_date` | DD/MM/YYYY หรือ DD/MM/YYYY (พ.ศ.) |
| `สินค้า` | `product_type` | ชนิดสินค้า เช่น Wood chips, ไม้สับ |
| `จำนวนพ่วง` | `truck_count` + `truck_type=พ่วง` | 1 พ่วง = 30 ตัน |
| `จำนวนสิบล้อ` | `truck_count` + `truck_type=สิบล้อ` | 1 สิบล้อ = 15 ตัน |
| `ปลายทาง` | `destination` | optional — ถ้าไม่มีใช้ชื่อลูกค้าแทน |
| `ต้นทาง` | `origin` | optional |

---

## น้ำหนักต่อคัน

| ประเภทรถ | น้ำหนัก |
|---|---|
| พ่วง | 30 ตัน |
| สิบล้อ | 15 ตัน |

ระบบจะคำนวณ `weight_ton = truck_count × tonPerTruck` อัตโนมัติ

---

## Fields ที่ระบบเพิ่มอัตโนมัติ

| Field | ค่า |
|---|---|
| `id` | P-xxxxxxxxxx (timestamp) |
| `status` | `pending` (รอคอนเฟิร์ม) |
| `reporter` | LINE userId ของผู้แจ้ง |
| `created_at` | เวลาที่รับข้อความ (วันที่ register) |

---

## Regex Patterns (backend/index.js → parsePlanText)

```javascript
ship_date    = /วันที่[^\n\d]*(\d{1,2}\/\d{1,2}\/\d{4})/
customer     = /ชื่อลูกค้า[:\s]+(.+)/  หรือ  /(?!วันที่).*ลูกค้า[:\s]+(.+)/
product_type = /สินค้า[:\s]+(.+)/
truck_count  = /จำนวนพ่วง[:\s]*(\d+)/  หรือ  /จำนวนสิบล้อ[:\s]*(\d+)/
destination  = /ปลายทาง[:\s]+(.+)/
origin       = /ต้นทาง[:\s]+(.+)/
```

## ตัวอย่าง Format ที่ระบบรับได้

**แบบมาตรฐาน (แนะนำ)**
```
ชื่อลูกค้า : Betagro ลพบุรี
วันที่จัดส่งลูกค้า : 16/6/2026
สินค้า : Wood chips ไม้สับ
จำนวนพ่วง : 2
```

**แบบมีสิบล้อ**
```
ชื่อลูกค้า : SCG บางซื่อ
วันที่จัดส่งลูกค้า : 20/6/2026
สินค้า : ไม้สับ
จำนวนสิบล้อ : 3
```

**แบบมี ต้นทาง/ปลายทาง**
```
ต้นทาง : โรงงาน A
ปลายทาง : Betagro ลพบุรี
วันที่จัดส่งลูกค้า : 16/6/2026
สินค้า : Wood chips
จำนวนพ่วง : 1
```
